    public addSegment(id: string, start: number, end: number, options: Object) source

    Add new segment to scope.

    Params:

    NameTypeAttributeDescription
    id string

    If of this segment.
    start number

    Index of first character of this segment.
    end number

    Index of first character after this segment.
    options Object

    Optional parameters.
    options.uses Object[]
    • optional

    Array of other segments which should be considered as part of this segment.
    options.usesPointers string[]
    • optional

    Array of pointers needed for this segment. Basically this means variables which this segment is using.
    options.pointer string
    • optional

    If this segment is going to create a new variable this would be the pointer to it.
    options.label string
    • optional

    If this segment is going to create a new label this would be the pointer to it.
    options.exposes string[]
    • optional

    If this segment is going to expose something outside of scope.

    Example:

    scope.addSegment(createId('VariableDeclaration'), 10, 33);
